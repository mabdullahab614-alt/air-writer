## 🎮 Controls Quick Reference

### Air Writer
| Input | Action |
|-------|--------|
| ☝️ 1 finger | Draw |
| ✌️ 2 fingers | Move |
| 🖐️ All fingers | Erase |
| C key | Clear |
| S key | Save |
| Q key | Quit |

### Air PC Controller
| Input | Action |
|-------|--------|
| ☝️ Index up | Move cursor |
| 👌 Thumb+Index pinch | Left click |
| 🤏 Thumb+Middle pinch | Right click |
| 🖐️ All fingers up | Scroll up |
| 👊 Fist | Scroll down |
| ✌️ 2 fingers close | Double click |
